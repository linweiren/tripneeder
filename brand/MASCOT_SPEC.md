# TripNeeder 吉祥物規格

## 吉祥物名稱
奶油白旅行貓

## 核心定位
TripNeeder 的吉祥物是一隻溫柔、好奇、可靠的旅行貓，代表「快速、可愛、文青、低負擔」的即興行程工具感。

它不是主功能本身，而是陪伴使用者在空狀態、載入、引導、完成行程等時刻降低壓力，讓產品更親切。

## 視覺特徵

### 主體
- 奶油白 / 米白色貓
- 整體低飽和、柔和、水彩感
- 不可使用高飽和卡通色
- 不可做成誇張迷因風或動漫熱血風

### 固定識別
必須保留：
- 淡橘耳尖
- 尾巴尾端淡橘 / 淺棕色塊
- 淡暖橘臉頰暈染
- 鼠尾草綠背包
- 卡其色帽子
- 小棕色項圈或吊牌

## 品牌氣質
- 快速：協助使用者馬上出發
- 可愛：降低使用壓力
- 文青：有手帳、城市散步、咖啡店、地圖筆記感
- 工具感：不能太裝飾化，仍要清楚好用
- 低飽和：整體色調要柔和、耐看

## 建議使用場景
- 首頁 Hero 區塊
- 空狀態畫面
- 載入中畫面
- 行程生成完成畫面
- 路線引導提示
- 收藏 / 分享成功提示

## 不建議使用場景
- 不要讓吉祥物搶走主要 CTA
- 不要讓吉祥物蓋過行程內容
- 不要在每張卡片都塞吉祥物，避免干擾資訊閱讀
- 不要使用過度誇張的表情與動作

## 品牌配色建議
- 奶油白：#FFF7EB
- 暖米：#F4E8D6
- 淡橘：#F2C9A8
- 淺棕：#CDAE8B
- 鼠尾草綠：#A7B59A
- 霧灰綠：#7E8F82
- 主文字深綠灰：#2F3A32

## 前端實作原則
- 展示型吉祥物優先使用透明背景 PNG / WebP
- 小 icon 或簡化角色才使用 SVG
- 吉祥物圖片放在 `src/assets/mascot/`
- 品牌參考資料放在 `docs/brand/`
- 圖片需加上合理 alt，例如 `TripNeeder 奶油白旅行貓吉祥物`
